# ocr-tesseract-demo-project
OCR Tesseract Demo Java Project 


Plus aspects:
OCR Tesseract is open source.
PDF, TIF, JPEG documents to Text parse is doing.

Negative aspects:
There are problems in Turkish and Russian alphabet characters.
